## Stack

- **Design tool:** [Pencil](https://pencil.evolveui.com/)
- **File format:** `.pen` (read/write via Pencil MCP)
- **Design tokens:** CSS custom properties / Pencil variables
- **Typography:** DM Sans, Inter, Playfair Display, Libre Baskerville, and others
- **Target platforms:** WordPress FSE / Gutenberg blocks, static HTML/Tailwind

## Usage

Open `.pen` files in the Pencil app or access via Pencil MCP tools in Claude Code.
